# Security

## Reporting a vulnerability

Please do **not** report security vulnerabilities via public GitHub issues.

If you believe you have found a security issue, please report it by email to the maintainers, via a private channel specified on [Kiploks](https://kiploks.com), or on [Discord](https://discord.com/users/1301837876987822145). Include a clear description and steps to reproduce if possible.

We will acknowledge receipt and work with you to understand and address the issue.

## Supported versions

Security updates are provided for the latest release. Upgrade to the newest version to receive patches.
